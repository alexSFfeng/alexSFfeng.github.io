About me:

INTRODUCTION:

Project Bounty
Webreg Arena Planner

Main Idea: UCSD webreg extension that collects current school year statistics and generate possible class schedules for target student.
Specification: Each generated schedules contain a calculated chance of student getting that schedule for his/her pick time.
Target Inconvenience: Help student pick classes in a wiser and more organized manner.

The Interviewer

Main Idea: Mobile application that allows user to match with other people for mock interview. There would be topics selection and scoring system
Specification: User could declare what major he/she is and what kind of topic he/she wants to get interview on. The system should have a default interviewer just in case there is no real person match
Target Inconvenience: Provide people a source for getting interview practice; thus increase their chances of getting a job or internship.

Note Reader

Main Idea: Mobile app for note taking. Take a picture of the text that needs to be scanned and enter keywords for topic and subject.
Specification: App should generate a brief summary base on user input and will organize key points that user should take note of
Target Inconvenience: Allow people to quickly find the key points of a text. Good for both studying and reading long and cluster text files.

Trip Guidance

Main Idea: Mobile/Desktop application for trip planning. Users specify the budget and place they want to go to. They will be match with users that are familiar with that place and had planned a trip there before.
Specification: App should contain translation functionality so that language barrier could be minimized. There should be an interface for timeline planning and cost calculation.
Target Inconvenience: Great way for cost efficient and time-saving traveling; it will also help people meet people from other countries.

Resources For Learning CS (MOSTLY FREE)

  • Codecademy would be a great start for learning some beginner friendly languages and techniques. What is great about it is that it is totally free and highly interactive.
  • W3Schools is very popular for learning web design. I would recommend learning HTML, CSS, JavaScript, and server side knowledge there.
  • SoloLearn is for people who would like to learn on the go. They have many mobile educational apps that teaches multiple programming language. The user interface looks clean and it is one of the mobile learning resource I would choose first for learning on the go.
  • EDX, MIT Open Courseware, and Stanford Lagunita is helpful in learning important CS concepts and algorithmic thinking. They have online courses that you can take for free and are mostly self-paced
